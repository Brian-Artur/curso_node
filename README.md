# Curso de NodeJS

## Propósito del repositorio
Este es un repositorio para estudiar en profundidad NodeJS para backend. Aquí 
pienso recopilaar la información nueva y hacer pruebas.

La idea es probar a hacer proyectos con Fastify porque veo que maneja las 
peticiones el doble de rápido que Express, y porque tiene mayor predisposición
a trabajar con TS en OOP.
